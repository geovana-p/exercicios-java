#Exercicios de java
Repositorio com exercicios para estudar logica da programação e java
## conteúdos
###01 - lógica básica
- mostra variaveis
- somar dois números
- converter anos em dias
- calcular a media de tres números
###02 - condicionais
- verificar se um número é positivo ou negativo
- verificar se um número é par ou ímpar
- soma se os numeros forem iguais e multiplica se forem diferentes
###Objetivo
praticar conceitos basicos de programação
